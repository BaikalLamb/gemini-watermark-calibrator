# Gemini 水印校准器
![Uploading 620743722-f9e13fef-c793-479c-9bbd-48ae4e4001f8.png…]()

Gemini 水印校准器是一个 Windows 图形化小工具，用于对 Gemini 图片右下角的星形水印区域进行位置校准、Gain 处理和黑边柔化。它适合处理固定位置、固定形态的图片水印场景，并提供局部放大预览，方便在导出前确认效果。

> 请只处理你自己拥有版权，或已经获得授权可以修改的图片。本工具不鼓励用于规避版权、署名、平台规则或其他合法标识。

## 功能特点

- 自动识别并校准右下角星形水印的大致位置和尺寸
- 支持手动拖动校准点，便于微调水印位置
- 支持 50%-800% 缩放查看局部细节
- 支持自定义 Gain 数值，范围为 0.30-0.80
- 支持黑边柔化，边缘宽度范围为 0-8 px
- 支持模糊强度调节，范围为 0.0-3.0
- 支持原图和处理结果预览
- 支持单独打开实时前后对比子窗口
- 支持导出 RGBA PNG 图片
- 内置自适应背景图和右下角署名 `By：听友小辰`

## 下载使用

1. 打开 GitHub Releases 页面
2. 下载 `GeminiWatermarkCalibrator-v0.1.1.exe`
3. 双击运行程序
4. 点击上传图片
5. 在右下角局部预览里确认自动校准位置
6. 如有偏差，放大图片后拖动校准标记进行微调
7. 调整 Gain、黑边宽度和模糊强度
8. 在预览窗口确认效果
9. 导出处理后的 PNG 图片

## 推荐参数

默认可以先从以下参数开始：

```text
Gain: 0.58
黑边宽度: 2-4 px
模糊强度: 1.0-2.0
```

如果水印残留偏亮，可以适当降低或提高 Gain 观察变化。  
如果星形轮廓周围仍有明显黑边，可以适当增加黑边宽度和模糊强度。  
如果细节被抹得太软，则降低模糊强度或缩小黑边宽度。

## 注意事项

本工具主要针对固定在图片右下角的 Gemini 星形水印区域。不同图片、不同背景、不同压缩程度下，处理效果会有差异。

对于背景纹理复杂、亮暗变化剧烈、压缩痕迹明显的图片，可能无法完全消除痕迹。此时建议使用手动校准、局部预览和较温和的模糊参数，优先获得自然观感。

程序会导出新图片，不会覆盖原图。正式处理前仍建议保留原始文件。

## 技术原理

程序使用固定水印模板和图像局部匹配，先在右下角区域估计水印位置，再对目标区域进行反向混合近似处理。Gain 参数用于调整去水印强度，黑边柔化用于弱化星形边缘附近较明显的暗色轮廓。

这个方法不是 AI 生成式修复，也不是对任意水印都通用的抹除工具。它更接近一个针对固定水印样式的校准和局部图像处理工具。

## 自行运行源码

需要 Python 3.12 或兼容版本。

安装依赖后运行：

```powershell
python app.py
```

主要依赖：

```text
PySide6
NumPy
OpenCV
Pillow
```

## 打包 exe

本项目使用 PyInstaller 打包。

```powershell
.\build.ps1
```

或直接运行：

```powershell
pyinstaller --noconfirm --clean WatermarkCalibrator.spec
```

打包后的 exe 通常位于：

```text
dist/Gemini水印校准器-v0.1.1.exe
```

## 第三方组件

本项目使用以下第三方组件：

- PySide6 / Qt for Python：桌面图形界面
- NumPy：数组和数值处理
- OpenCV：图像匹配、模糊和局部处理
- Pillow：图片读取和保存辅助
- PyInstaller：Windows exe 打包

水印透明度模板参考了 `likemaoke/banana-watermark-remover` 相关实现思路，该项目 README 中标注 MIT License，并提到来源参考 `GargantuaX/Gemini-NanoBanana-Watermark-Remover`。
